# sitorama

#### Установка:
```bash
git clone https://github.com/art-sitedesign/sitorama.git
cd sitorama
```

#### Сборка приложения:
```bash
make app.build
```

#### Запуск приложения:
- для macos:
```bash
make app.run.mac
```
- для linux:
```bash
make app.run.linux
```
